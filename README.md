substring_elimination
=====================

A small algorithm for substring elimination

input with two strings: X and Y
output the minimun number of charcters taken from X to make Y not the substring of X

example:

X = 123     Y = 12
min: 1

X = 123123  Y = 123
min: 2

X = 123123123 Y = 123123
min: 1

X = 1111222   Y = 12
min: 3
